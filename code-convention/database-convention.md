# Database Convention

## Table

Table names should be in singular, all small case and words separated by `_` (underscore)

**Don't**

```
users
```

```
timeSlots
```

```
timeSlot
```

**Do:**

```
user
```

```
time_slot
```

## Column names

Column names should be in all small cases and words separated by `_` (underscore)

**Don't**

```
Id
```

```
firstName
```

**Do:**

```
id
```

```
first_name
```
